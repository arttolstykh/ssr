# SSR

Server-side rendering (SSR) with [Puppeteer](https://github.com/puppeteer/puppeteer) (proof of concept based on the [article](https://developers.google.com/web/tools/puppeteer/articles/ssr)).

## Install:

```bash
npm install
```

## Running:

```bash
npm start
```

### Development:

```
npm run develop
```
